Dato adicional

HTML es un lenguaje de marcado que se utiliza para crear la estructura de una página web. Es fundamental para el desarrollo web y es el primer paso para aprender a crear sitios web. Con HTML, puedes definir elementos como encabezados, párrafos, enlaces, imágenes y mucho más, lo que te permite construir páginas web atractivas y funcionales.
